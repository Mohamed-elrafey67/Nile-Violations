# منظومة رصد التعديات على نهر النيل
## نظام Django متعدد المحافظات

---

## متطلبات التشغيل
```
Python >= 3.10
pip install -r requirements.txt
```

---

## الإعداد الأول

```bash
# 1. إنشاء قاعدة البيانات
python manage.py migrate

# 2. نسخ ملفات الخرائط
mkdir -p data/
cp egypt_adm3_geo.json data/
cp egypt_govs.json     data/

# 3. استيراد بيانات الدقهلية
python manage.py import_violations الدقهلية_كاملة.xlsx \
    --geo-json data/egypt_adm3_geo.json

# 4. إنشاء مستخدم الإدارة (اختياري)
python manage.py createsuperuser

# 5. تشغيل التطبيق
python manage.py runserver
```

ثم افتح: http://127.0.0.1:8000

---

## إضافة محافظة جديدة في المستقبل

```bash
# استيراد محافظة جديدة — لا يحذف بيانات الدقهلية
python manage.py import_violations الشرقية_كاملة.xlsx \
    --geo-json data/egypt_adm3_geo.json

# استيراد عدة محافظات دفعة واحدة
python manage.py import_violations data/*.xlsx \
    --geo-json data/egypt_adm3_geo.json

# إعادة استيراد محافظة معينة بعد تحديث البيانات
python manage.py import_violations الدقهلية_كاملة.xlsx \
    --clear-gov EG12 \
    --geo-json data/egypt_adm3_geo.json

# تجربة الاستيراد بدون حفظ
python manage.py import_violations الشرقية.xlsx \
    --geo-json data/egypt_adm3_geo.json \
    --dry-run
```

## إضافة تصحيحات أسماء لمحافظة جديدة

أنشئ ملف: `data/name_corrections/<PCODE>.json`

مثال لمحافظة الشرقية (`EG13.json`):
```json
{
  "منشية ابو عمر": "منشأة أبو عمر",
  "كفر صقر البلد": "كفر صقر"
}
```

سيُحمَّل تلقائياً عند استيراد بيانات هذه المحافظة.
